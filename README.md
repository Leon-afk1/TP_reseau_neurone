# TP_reseau_neurone

## MiniProjet - Installation et préparation

Ce projet nécessite la création d’un environnement virtuel, l’installation des dépendances et le téléchargement manuel du dataset. Suivez les étapes ci-dessous.

---

### 1. Créer un environnement virtuel

À la racine du projet, exécuter :

```bash
python3 -m venv venv
```

Activer ensuite l'environnement, sur Linux/Mac :

```bash
source venv/bin/activate
```

Sur Windows :

```bash
venv\Scripts\activate
```

### 2. Installer les dépendances

Avec l’environnement virtuel activé, installer les dépendances :

```bash
pip install -r requirements.txt
```

### 3. Téléchargement du dataset

Télécharger le dataset pour le mini-projet à l’adresse suivante : https://erda.ku.dk/public/archives/ff17dc924eba88d5d01a807357d6614c/TrainIJCNN2013.zip.

Plcaer ensuite le dossier décompressé `TrainIJCNN201` dans le dossier `MiniProjet/`.
